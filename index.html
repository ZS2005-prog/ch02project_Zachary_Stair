Building HTML Lists

Embedded style sheet
Styling information that is written on the top part of a HTML page.
External style sheet
Text file that is separate from the HTML file that stores the CSS style information for a website
Inline style
Styling information for an HTML tag that is written inside of, that is, inline with, the HTML tag as a set of attributes
